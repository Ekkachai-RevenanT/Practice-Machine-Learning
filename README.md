# Tools ที่ใช้
-Python (Pandas and Scikit-learn)

-Logistic & K-Nearest Neighbor(Standard) : เป็นไฟล์แบบฝึกหัดที่ผมได้ทำแต่ใช้ Scaler เป็น Standard
